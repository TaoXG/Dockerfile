BookLogr 前端页面。


`BookLogr` 源码地址：[https://github.com/Mozzo1000/booklogr](https://github.com/Mozzo1000/booklogr)

构建镜像和容器运行的基本命令如下👇

```bash
# 拉取源代码
git clone https://github.com/Mozzo1000/booklogr.git

# 进入代码目录
cd booklogr/web

# 将 Dockerfile 放入当前目录

# 构建镜像
docker build -t wbsu2003/booklogr-web:v1 .
```
